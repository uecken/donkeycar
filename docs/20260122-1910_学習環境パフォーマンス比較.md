## 4. 環境別メリット・デメリット

### NVIDIA GPU（GTX 1660 Ti / RTX 3060等）

| メリット | デメリット |
|---------|-----------|
| ✅ 最高の学習速度 | ❌ 消費電力が高い |
| ✅ CUDA/cuDNN最適化 | ❌ デスクトップPC必要 |
| ✅ 豊富な情報・サポート | ❌ WSL2設定がやや複雑 |
| ✅ 中古でコスパ良好 | ❌ 携帯性なし |

**推奨用途**: 頻繁な学習、大量データ、本格開発

### Mac M1/M2シリーズ

| メリット | デメリット |
|---------|-----------|
| ✅ セットアップ簡単 | ❌ 学習速度は中程度 |
| ✅ 低消費電力 | ❌ CUDA非対応 |
| ✅ 高い携帯性 | ❌ 一部ライブラリ非対応の場合あり |
| ✅ 静音 | ❌ メモリ増設不可 |

**推奨用途**: 軽量な学習、開発・デバッグ、出先作業

### CPU（Intel/AMD）

| メリット | デメリット |
|---------|-----------|
| ✅ 追加ハードウェア不要 | ❌ 学習速度が非常に遅い |
| ✅ 環境構築シンプル | ❌ 大量データに不向き |
| ✅ どの環境でも動作 | ❌ 実用的でない |

**推奨用途**: 動作確認のみ、少量データのテスト

### Google Colab

| メリット | デメリット |
|---------|-----------|
| ✅ 無料でGPU利用可能 | ❌ セッション時間制限（12時間） |
| ✅ 環境構築不要 | ❌ ネットワーク必須 |
| ✅ T4 GPU（16GB VRAM） | ❌ データ転送が必要 |
| ✅ どこからでもアクセス | ❌ 長時間学習に不向き |

**推奨用途**: 初心者、一時的な利用、トライアル

## 6. 推奨構成

### 初心者・趣味向け

```
Google Colab（無料）
  ↓ 物足りなくなったら
GTX 1660 Ti（中古 2-3万円）+ WSL2
```

### 本格開発向け

```
RTX 3060（12GB）+ Ubuntu or WSL2
  - 大量データ対応
  - 複数モデル同時学習可能
```

### モバイル開発向け

```
MacBook Pro (M1 Pro/M2 Pro)
  - 出先での開発・軽量学習
  - メイン学習はColab/自宅サーバー併用
```
